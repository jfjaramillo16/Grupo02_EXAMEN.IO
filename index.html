Slider - Grupo 02

1. Teoría
El Slider permite mostrar imágenes o textos de manera dinámica.
Bootstrap y jQuery facilitan la creación de efectos automáticos y navegación entre elementos.

Un Slider mejora la presentación de contenido visual sin recargar la interfaz.

2. Ejemplo (Video + Explicación)
<div id="miSlider" class="carousel slide">
  <div class="carousel-inner">
    <div class="carousel-item active">
      <img src="img/imagen1.jpg">
    </div>
  </div>
</div>

El código anterior muestra la estructura básica del carrusel con clases .carousel y .carousel-item.

3. Ejecución del Código
Implementación del Slider funcional de Bootstrap con controles de navegación y captions.

4. Ejercicio (Con Efecto / Animación)
$(document).ready(function(){
  let index = 0;
  const imgs = $('#sliderFade .fade-img');
  setInterval(() => {
    imgs.eq(index).fadeOut(1000);
    index = (index + 1) % imgs.length;
    imgs.eq(index).fadeIn(1000);
  }, 3000);
});
